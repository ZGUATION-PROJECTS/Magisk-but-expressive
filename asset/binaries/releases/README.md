# Release binaries

This directory stores extracted Magisk Alpha native binaries by release.

Each release uses this layout:

```text
<version>-<versionCode>/
  manifest.json
  arm64-v8a/
    busybox
    init-ld
    magisk
    magiskboot
    magiskinit
    magiskpolicy
  armeabi-v7a/
  x86/
  x86_64/
```
